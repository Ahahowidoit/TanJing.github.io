1 绘制标准正态分布图

1.1 问题

绘制正态分布图。

1.2 步骤

实现此案例需要按照如下步骤进行。

1）导入需要的环境库

2）创建一组标准正态分布的数据 data1接收

3）绘制直方图：划分为不同的区间，统计不同区间频数

4）绘制箱线图：查看数据分布范围

5）绘制密度曲线图：连续型变量概率分布图

6）分析数据:均值+-1倍 2倍 3倍 标准差比例各是多少（对于data1数据分段采用cut函数）

1.3 代码

完整代码如下：

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
# 忽略警告
import warnings
warnings.filterwarnings('ignore')
# 创建一组标准正态分布的数据
# 均值0 标准差1
np.random.seed(123)
data1=np.random.randn(10000)
data1
s1=pd.Series(data1)
# s1
s1.mean()
s1.std()
s1.plot(kind='hist')
s1.plot(kind='kde')
plt.xlim(-5,5)
plt.xticks(np.arange(-5,6,1))
plt.show()
s1.describe()
cut1=np.arange(-4,5,1)
cut1
cut_data1=pd.cut(s1,cut1)
cut_data1.value_counts().sort_index(axis=0)

2 求区间占比

2.1 问题

初三年级学生 1000名 语文成绩平均分80 标准差5，是一组符合正态分布的数据，求：75-85分占比是多少？

2.2 步骤

实现此案例需要按照如下步骤进行。

1）data2接收，随机种子123随机数，生成1000个符合正态分布的数据

2）绘制直方图 绘制密度曲线图

3）验证均值与标准差倍数的分布

4）求75-85分占比是多少代码

2.3 代码

完整代码如下：

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
# 忽略警告
import warnings
warnings.filterwarnings('ignore')
np.random.seed(123)
data2=np.random.normal(loc=80,scale=5,
                      size=1000)
# data2
s2=pd.Series(data2)
# s2
s2.plot(kind='hist')
s2.plot(kind='kde')
plt.xlim(60,100)
plt.show()
s2.describe()
cut2=np.arange(60,100,5)
cut2_data=pd.cut(s2,cut2)
cut2_data.value_counts().sort_index(axis=0)
# 导入高级科学计算库统计 概率函数
import scipy.stats
# cdf:累积分布函数  计算<=某个值的概率
scipy.stats.norm.cdf(75,loc=80,scale=5)
# cdf:累积分布函数  计算<=某个值的概率
scipy.stats.norm.cdf(85,loc=80,scale=5)
