4 变量和赋值语句综合练习

4.1 问题

写一个程序 myprog.py

  1. 让用户输入用户名
  2. 让用户输入密码
  3. 打印 谁谁谁 的密码是 xxx

4.2 方案

编写程序时，很多情况下都需要程序与用户交互。在python3中，主要通过input()获取用户输入信息，使用print()打印信息。使用赋值语句可以创建变量用于临时记住表达式执行的结果。

4.3 步骤

实现此案例需要按照如下步骤进行。

步骤一：编写程序

本次练习的程序文件是myprog.py。写入读取用户名和密码的语句

username = input('请输入用户名: ')     # 使用变量username接收用户输入的字符
password = input('请输入密码: ')       # 使用变量password接收用户输入的密码字符

print(username, '的密码是', password)  # 输出最后想要的信息，字符串和变量名之间用逗号分隔开

步骤二：测试脚本执行

在PyCharm 中右键点击程序，选择 Run ‘myprog’ . 运行程序并输入用户名和密码

请输入用户名: root
请输入密码: 123456
root 的密码是 123456
