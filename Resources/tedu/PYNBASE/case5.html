1 面向对象

1.1 问题

写一个小狗类，大致框架如下

class Dog:
    def __init__(self, ...):
        pass

实现以下的功能

dog1 = Dog('白色', '藏獒')
dog1.eat(1, '羊肉')   # 白色 的 藏獒 吃了 1 斤 羊肉,
dog2 = Dog('灰色', '导盲犬')
dog2.eat(2, '狗粮')   # 灰色 的 导盲犬  吃了 2 斤 狗粮,

dog1.eat(2, '牛肉')   # 白色 的 藏獒 吃了 2 斤 牛肉,
dog1.info()   # 白色 的 藏獒 吃过 ['羊肉', '牛肉']
dog2.info()   # 灰色 的 导盲犬 吃过 ['狗粮']

1.2 方案

类用来描述一个对象的属性和行为。从调用方法，我们可以看出，应当是有三种方法,分别为初始化方法__init__、吃的方法eat、显示信息的方法info；从调用的参数传递可以看出Dog类型的对象有如下属性:颜色color、类别kind、吃过的食物foods。

根据创建对象的语句 dog1 = Dog('白色', '藏獒') 可以看出初始化方法应当额外传入两个参数,我们取名为c 和k 则初始化方法的定义方法可以确定为 def __init__(self, c, k):。因为有上述三个属性需要保存，所以在初始化方法内可以创建此对象的三个属性，代码如下：

class Dog:
    def __init__(self, c, k):
        self.color = c  # 小狗的颜色
        self.kind = k  # 小狗的种类
        self.foods = []  # 小狗吃过的食物

根据小狗吃的行为的方法：dog1.eat(1, '羊肉')，可以看出此方法需要传入两个额外的变量用来绑定吃的质量weight和吃的食物food。同时此方法访问自己的颜色和种类用于显示信息，因此，此eat 方法的实现代码可以编写如下：

    def eat(self, weight, food):
        print(self.color, '的', self.kind, '吃了', weight, '斤', food)

因为在显示信息中还要显示小狗吃过的食物，因此还要将food 绑定的食物放入到此小狗自己记录食物的列表 foods 中。因此在eat方法内还要加入如下语句

        self.foods.append(food)  #  将吃过的食物放入列表中

根据小猴显示信息的方法dog2.info() 可以看出，显示信息只是使用小狗自身的属性，不需要传入其他的信息，因此info方法内部只需要传入一个调用此方法的对象的self参数即可。因此方法可以用如下代码创建

     def info(self):
        print(self.color, '的', self.kind, '吃过', self.foods)

1.3 步骤

实现此案例需要按照如下步骤进行。

步骤一：使用PyCharm 编写程序文件

打开PyCharm， 创建一个新的项目(New Project)，在项目文件夹中创建myprog1.py 文件。

步骤二：使用PyCharm 编写程序文件

按如下示例编写文件

class Dog:
    def __init__(self, c, k):
        self.color = c  # 小狗的颜色
        self.kind = k  # 小狗的种类
        self.foods = []  # 小狗吃过的食物

    def eat(self, weight, food):
        print(self.color, '的', self.kind, '吃了', weight, '斤', food)
        self.foods.append(food)  #  将吃过的食物放入列表中

    def info(self):
        print(self.color, '的', self.kind, '吃过', self.foods)

dog1 = Dog('白色', '藏獒')
dog1.eat(1, '羊肉')   # 白色 的 藏獒 吃了 1 斤 羊肉,
dog2 = Dog('灰色', '导盲犬')
dog2.eat(2, '狗粮')   # 灰色 的 导盲犬  吃了 2 斤 狗粮,

dog1.eat(2, '牛肉')   # 白色 的 藏獒 吃了 2 斤 牛肉,
dog1.info()   # 白色 的 藏獒 吃过 ['羊肉', '牛肉']
dog2.info()   # 灰色 的 导盲犬 吃过 ['狗粮']

步骤三：运行程序myprog1.py

在PyCharm 中右键点击myprog1.py 文件，选择 run ‘myprog1’ 运行程序，示例如下:

灰色 的 导盲犬 吃了 2 斤 狗粮
白色 的 藏獒 吃了 2 斤 牛肉
白色 的 藏獒 吃过 ['羊肉', '牛肉']

2 安装模块

2.1 问题

1）安装numpy、pandas、matplotlib模块

2）使用管理员权限运行命令提示符

3）在管理员权限下的命令提示符下安装 jupyter 模块

2.2 方案

安装python 的第三方模块和包可以在连接互联网的情况下使用 pip 命令在线安装。命令的格式为：

pip install 包名或模块名

需要注意的是 pip 默认使用国外的源（用于存放数据的服务器）来下载第三方包，速度比较慢，因此可以使用pip 命令提供的 -i 选项后跟或内的源服务器，在国内下载数据包会快很多。国内的源有:

清华大学 ：https://pypi.tuna.tsinghua.edu.cn/simple/
阿里云：http://mirrors.aliyun.com/pypi/simple/
中国科学技术大学 ：http://pypi.mirrors.ustc.edu.cn/simple/
华中科技大学：http://pypi.hustunique.com/
豆瓣源：http://pypi.douban.com/simple/
腾讯源：http://mirrors.cloud.tencent.com/pypi/simple
华为镜像源：https://repo.huaweicloud.com/repository/pypi/simple/

使用指定源的命令格式为

pip install 包名或模块名 -i 源连接地址

另外，有些包需要使用管理员权限才能安装。此时需要 以管理员身份运行 命令提示符（终端），如jupyter。
